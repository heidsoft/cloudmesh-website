📊 数据分析设置指南

1. Google Analytics 设置

替换代码中的 G-XXXXXXXXXX 为您的 Google Analytics 跟踪 ID

  1. 访问 Google Analytics
  2. 创建账号和媒体资源
  3. 获取跟踪 ID（格式：G-XXXXXXXXXX）
  4. 替换代码中的跟踪 ID

2. 百度统计设置

替换代码中的 xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx 为您的百度统计 ID

  1. 访问 百度统计
  2. 注册账号并添加网站
  3. 获取统计代码
  4. 替换代码中的统计 ID

3. 转化跟踪设置

试用申请跟踪

gtag('event', 'conversion', {
  'send_to': 'AW-XXXXXXXXX/YYYYYYYYYY'
});

演示预约跟踪

gtag('event', 'schedule_demo', {
  'event_category': 'engagement',
  'event_label': 'Demo Request'
});

4. 数据报表

建议监控的核心指标：

  • 每日访问量（UV/PV）
  • 流量来源渠道
  • 页面停留时间
  • 跳出率
  • 转化率（试用/演示/联系）
