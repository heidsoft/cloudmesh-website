# Contributing to CloudMesh

感谢您对 CloudMesh 项目的兴趣！🎉

我们欢迎各种形式的贡献，包括但不限于：
- 🐛 Bug 报告
- 💡 功能建议
- 📝 文档改进
- 🔧 代码提交
- 📖 博客文章

## 🚀 开发流程

### 1. Fork & Clone

```bash
# Fork 后克隆到本地
git clone https://github.com/YOUR_USERNAME/cloudmesh-website.git
cd cloudmesh-website
```

### 2. 创建分支

```bash
# 功能分支命名规范
git checkout -b feature/your-feature-name
# 或
git checkout -b fix/your-bug-fix
```

### 3. 开发 & 提交

```bash
# 本地预览
python3 -m http.server 8080

# 提交格式
git add .
git commit -m 'Add: your feature description'
git commit -m 'Fix: your bug fix description'
```

### 4. Push & Pull Request

```bash
# 推送到你的 Fork
git push origin feature/your-feature-name
```

然后在 GitHub 上创建 Pull Request。

## 📝 博客写作规范

1. 博客文件放在 `blog/` 目录
2. 使用 HTML 格式，参考现有博客结构
3. 必须包含的 meta 信息：
   - `title`
   - `description`
   - `keywords`
   - `author`
   - 发布日期
4. 图片使用 CDN 或相对路径引用

## ✅ Pull Request 检查清单

- [ ] 代码格式规范
- [ ] 提交信息清晰
- [ ] 添加了必要的测试（如适用）
- [ ] 更新了相关文档（如适用）
