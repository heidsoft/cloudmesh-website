核心功能

📋
事件管理

快速响应和处理 IT 服务中断，支持多级升级和 SLA 管理

🔄
变更管理

规范变更流程，降低变更风险，支持 RFC → CAB → 实施 → 评审

问题管理

根因分析，降低重复事件，支持知识库积累

🗄️
配置管理(CMDB)

统一管理 IT 资产和配置项，支持关系图和影响分析
