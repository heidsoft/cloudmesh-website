🚀 ITSM 快速入门指南

5 分钟快速完成部署，立即开始使用 IT 服务管理

预计时间：5 分钟

前提条件

  • ✅ Linux/macOS 或 Windows 系统
  • ✅ 已安装 Docker 和 Docker Compose
  • ✅ 能够访问互联网
  • ✅ 基础命令行操作知识

系统要求

最低配置：1核CPU / 1GB内存 / 10GB磁盘，推荐配置：2核CPU / 4GB内存

快速开始

  1. 克隆项目
    命令行
    git clone https://github.com/heidsoft/itsm.git cd itsm
  2. 启动服务
    Docker Compose
    docker-compose up -d
  3. 访问系统
    打开浏览器访问 http://localhost:8000

启动成功

默认管理员账号：admin / admin123

技术架构

💬 前端 Vue.js 3 + Element Plus
⚙️ 后端 Python FastAPI
🗄️ 数据库 PostgreSQL
🚀 缓存 Redis

核心功能

  • 事件管理 — 快速响应 IT 服务中断，支持 SLA 管理
  • 变更管理 — 规范变更流程，降低风险
  • 问题管理 — 根因分析，降低重复事件
  • 配置管理(CMDB) — 统一管理 IT 资产和配置项
  • 知识库 — 积累解决方案，减少重复劳动
