⚡ 5 分钟快速上手 OpenClaw

从 0 到 1 配置你的 AI 助手

2026-03-11 · 阅读时间：5 分钟

📖 前言

本教程将带你用 5 分钟时间完成 OpenClaw 的配置和使用。无论你是个人开发者还是企业用户，都能快速上手。

💡 前提条件

• 一台能上网的电脑（Linux/macOS/Windows）
• 钉钉/飞书/企业微信等平台账号
• 基础命令行操作知识

🎯 步骤详解

访问配置生成器（1 分钟）

打开浏览器，访问配置生成器页面：

https://cloudmesh.top/openclaw-generator/

选择通讯平台（1 分钟）

根据你的使用场景选择合适的平台：

  • 钉钉 - 企业 IM，适合国内企业
  • 飞书 - 企业协作平台
  • 企业微信 - 企业通讯工具
  • Telegram - 国际即时通讯
  • Discord - 社群聊天
  • Slack - 国际团队协作

💡 如何选择？

国内企业推荐钉钉/飞书/企业微信，国际团队推荐 Slack/Telegram。

填写平台凭证（1 分钟）

根据选择的平台，填写对应的应用凭证：

# 钉钉示例 AppKey: dingxxxxxxxxxx AppSecret: 你的应用密钥 # 飞书示例 App ID: cli_xxxxxxxxxx App Secret: 你的应用密钥 # 企业微信示例 BotID: 机器人 ID Secret: 应用密钥

🔒 安全提示

敏感信息只会保存在本地配置文件中，不会上传到服务器，请放心填写。

配置 Agent 团队（1 分钟）

选择你的 Agent 团队规模：

  • 👤 1 个 - 单兵作战（个人使用）
  • 👥 2 个 - 双人协作（小团队）
  • 👨‍💼 3 个 - 专业小队（中型团队）
  • 🏢 4 个 - 完整团队（大型团队）

生成并安装（1 分钟）

点击"🚀 生成配置"按钮，复制安装命令：

# Linux/macOS curl -fsSL "https://cloudmesh.top/openclaw-generator/install.sh" -o install.sh && bash install.sh # Windows (PowerShell) powershell -c "Invoke-RestMethod 'https://cloudmesh.top/openclaw-generator/install.ps1' -OutFile install.ps1; .\install.ps1"

打开终端，粘贴命令并回车，脚本会自动完成所有安装步骤。

✅ 验证安装

检查服务状态

openclaw status

如果看到以下输出，说明安装成功：

✅ OpenClaw 运行中 版本：2026.2.24 平台：dingtalk

测试消息

在钉钉/飞书等平台发送测试消息，OpenClaw 应该能够响应。
