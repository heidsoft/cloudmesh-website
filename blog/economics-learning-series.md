# 经济学学习系列：理解世界的底层逻辑

> "经济学研究的是人们如何利用稀缺的资源生产有价值的商品并分配给不同的人。" — N. Gregory Mankiw

---

## 为什么学经济学？

**经济学是理解世界的底层框架：**
- 个人决策（消费、工作、投资）
- 企业经营（定价、竞争、市场策略）
- 政府政策（税收、货币、贸易）
- 全球格局（汇率、通货膨胀、贫富差距）

## 核心概念速览

### 🔑 经济学的十大原理（曼昆）

**人们如何做出决策：**
1. 人们面临权衡取舍
2. 某种东西的成本是为了得到它所放弃的东西（机会成本）
3. 理性人考虑边际量
4. 人们会对激励做出反应

**人们如何相互影响：**
5. 贸易可以使每个人的状况变得更好
6. 市场通常是组织经济活动的好方法
7. 政府有时可以改善市场结果

**整体经济如何运行：**
8. 一国的生活水平取决于它生产物品与服务的能力
9. 当政府发行了过多货币时物价上涨（通货膨胀）
10. 社会面临通货膨胀与失业之间的短期权衡（菲利普斯曲线）

### 📊 供需模型

```
        价格
         │
    Pmax │─────────● 需求曲线 D
         │        ╱
         │       ╱
    Pe   │      ●───────── 均衡点 E
         │     ╱       ╱
         │    ╱       ╱
    Pmin │   ╱───────●
         │  ╱ 供给曲线 S
         │
         └──────────────── 数量 Q
              Qe
```

**核心公式：**
- 均衡：Qs = Qd
- 需求弹性：Ed = (%ΔQd) / (%ΔP)
- 供给弹性：Es = (%ΔQs) / (%ΔP)

### 💰 机会成本示例

**决策情境：**
- 你有 10 万元，可以：
  - 存款年利率 3%，年收益 3000 元
  - 投资店铺年收益 1.5 万元

**机会成本分析：**
- 选择存款：机会成本 = 1.5 万元（放弃的店铺收益）
- 选择开店：机会成本 = 3000 元（放弃的存款利息）
- **理性决策：开店**（收益更高）

### 🎮 博弈论入门：囚徒困境

**经典案例：**

两个嫌疑人被分别审讯：
- 都坦白：各判 5 年
- 都不坦白：各判 1 年
- 一人坦白另一人沉默：坦白者释放，沉默者判 10 年

**收益矩阵：**

|           | 对方坦白 | 对方沉默 |
|-----------|----------|----------|
| **我坦白** | -5, -5 | 0, -10 |
| **我沉默** | -10, 0 | -1, -1 |

**纳什均衡：双方都坦白**（尽管"都不坦白"对双方都更好）

**启示：** 个人理性与集体理性可能冲突 —— 这就是为什么需要制度与约束。
